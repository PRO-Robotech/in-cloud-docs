# 2.4. Data

## 2.4.1 Data Storage Management System Requirements

<table>
  <thead>
    <tr>
      <th>No.</th>
      <th>Requirement</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>The platform must support the use of storage systems with block data access</td>
    </tr>
    <tr>
      <td>2</td>
      <td>The platform must support the use of storage systems connected via FC protocol</td>
    </tr>
    <tr>
      <td>3</td>
      <td>The platform must support the use of storage systems connected via iSCSI protocol</td>
    </tr>
    <tr>
      <td>4</td>
      <td>The platform must support the use of storage systems connected via NFS protocol</td>
    </tr>
    <tr>
      <td>5</td>
      <td>The platform must support the use of storage systems connected via CIFS/SMB protocol</td>
    </tr>
    <tr>
      <td>6</td>
      <td>The platform must support connecting to storage systems without exclusive access to hardware</td>
    </tr>
    <tr>
      <td>7</td>
      <td>The platform must support direct volume passthrough to VMs via FC protocol</td>
    </tr>
    <tr>
      <td>8</td>
      <td>The platform must support direct volume passthrough to VMs via iSCSI protocol</td>
    </tr>
    <tr>
      <td>9</td>
      <td>The platform must support distributed data storage between remote sites based on storage systems with block data access (only for Master Nodes)</td>
    </tr>
    <tr>
      <td>10</td>
      <td>The platform must support vertical and horizontal storage system scaling without interrupting platform operation for platform components</td>
    </tr>
    <tr>
      <td>11</td>
      <td>The platform must support vertical and horizontal storage system scaling without interrupting platform operation for IaaS/PaaS/SaaS services</td>
    </tr>
    <tr>
      <td>12</td>
      <td>Storage systems within the Platform must support data encryption technologies</td>
    </tr>
    <tr>
      <td>13</td>
      <td>The platform must support dynamic allocation for VM disks (Thin Provisioning technology)</td>
    </tr>
    <tr>
      <td>14</td>
      <td>The platform must support fault-tolerant connection to storage systems (MPIO) for FC, iSCSI, S3</td>
    </tr>
    <tr>
      <td>15</td>
      <td>The platform must support S3 management: bucket size increase, bucket creation, backup</td>
    </tr>
  </tbody>
</table>
