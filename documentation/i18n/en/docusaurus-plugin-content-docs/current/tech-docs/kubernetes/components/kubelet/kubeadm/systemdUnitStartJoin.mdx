> This command starts the Kubelet service as part of a dedicated Kubeadm utility phase.

:::warning Note
This section depends on the following sections:
- [Kubeadm Configs](/tech-docs/kubernetes/components/kubeadm/lifecycleSettings.mdx).
:::

<h4>Start kubelet</h4>
<CodeBlock language="bash">
  {dedent`
    kubeadm join phase kubelet-start \\
      --config=${CUSTOM_VALUE.kubeadmBaseConfigPath.value}/kubeadm.yaml
  `}
</CodeBlock>
