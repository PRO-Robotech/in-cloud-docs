**Description**: Runc is a low-level tool for creating and managing containers, implementing the Open Container Initiative (OCI) specification. It is used for running containers in Kubernetes via CRI (Container Runtime Interface).

#### Functions
- Running containers using container images.
- Managing the container lifecycle (creation, startup, shutdown).
- Providing container isolation and security at the operating system level.

#### Features
- Works as a container runtime for Kubernetes, ensuring container execution in a strictly isolated environment.
- Interacts with the container runtime via CRI, using interfaces such as containerd or CRI-O.
