Конфигурация компонента

Конфигурация kubeadm для worker-узла описывает параметры JoinConfiguration (bootstrap-токен, apiServerEndpoint). Используется при подключении воркера к существующему кластеру.

Конфигурация Kubeadm
Конфигурация Kubeadm для подключения worker ноды к кластеру

Переменные окружения для шаблона конфигурационного файла

export CLUSTER_API_ENDPOINT=api.my-first-cluster.example.com
export BOOTSTRAP_TOKEN=fjt9ex.lwzqgdlvoxtqk4yw

Конфигурационный файл kubeadm для подключения воркера к кластеру

cat <<EOF > /var/run/kubeadm/kubeadm.yaml
---
apiVersion: kubeadm.k8s.io/v1beta3
discovery:
  bootstrapToken:
    apiServerEndpoint: ${CLUSTER_API_ENDPOINT}:6443
    unsafeSkipCAVerification: true
    token: "${BOOTSTRAP_TOKEN}"
kind: JoinConfiguration
EOF
