2.4. Данные

2.4.1 Требования к системе управления хранением данных​

№Требование
1Платформа должна поддерживать возможность использования СХД с блочным доступом к данным
2Платформа должна поддерживать возможность использования СХД, подключенных по протоколу FC
3Платформа должна поддерживать возможность использования СХД, подключенных по протоколу iSCSI
4Платформа должна поддерживать возможность использования СХД, подключенных по протоколу NFS
5Платформа должна поддерживать возможность использования СХД, подключенных по протоколу CIFS/SMB
6Платформа должна поддерживать подключение к СХД без использования монопольного доступа к оборудованию
7Платформа должна поддерживать прямой проброс томов в ВМ по протоколу FC
8Платформа должна поддерживать прямой проброс томов в ВМ по протоколу iSCSI
9Платформа должна поддерживать возможность организации распределенного хранения данных между удаленными площадками на базе СХД с блочным доступом к данным (только для Master Nodes)
10Платформа должна поддерживать вертикальное и горизонтальное масштабирование СХД без прерывания работы платформы для компонентов платформы
11Платформа должна поддерживать вертикальное и горизонтальное масштабирование СХД без прерывания работы платформы для сервисов IaaS/PaaS/SaaS
12СХД в составе Платформы должна поддерживать технологии шифрования данных
13Платформа должна поддерживать динамическое выделение для дисков ВМ (технология Thin Provisioning)
14Платформа должна поддерживать отказоустойчивое подключение к системе хранения данных (MPIO) для FC, iSCSI, S3
15Платформа должна поддерживать управление S3: увеличение размера бакета, создание бакетов, резервное копирование
